Role Definitions:

  • Source: High resolution quality file either digital or born digital. Used to generate service and thumbnail files, when relevant and needed. Current examples are TIFF,JPEG, WAV, MOV/AVI
  • Alternate: Another version of either a source or service file. Also used for files in non-standard formats (RAW images, proprietary authoring tools, etc.).
  • Service:The file provided to end users in user interface or for download. Either derived from source image or can be the only available file.
  • Preview/Thumbnail/Icon: Files used as proxies for the service file in the user interface.
